# Blackerz::Sdk

Ruby SDK to interact with blackerz bot list
## Installation

Add this line to your application's Gemfile:

```ruby
gem 'blackerz-sdk'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install blackerz-sdk

## Usage

```rb
data = Blackerz::Bots.bot_data("867571104784318464") # bot client id
p data
```
